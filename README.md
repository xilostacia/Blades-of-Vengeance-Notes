Notes & Usage

These notes are maintained as a running record of the campaign and are updated after each session.

They are written in Markdown format, which means:

You can open them with Notepad or any basic text editor
Everything will still be readable
Some structure and navigation will be limited
For the intended experience, use Obsidian
Automatically handles linking between entries
Provides graph view, backlinks, and navigation
Makes it easier to follow characters, locations, and events

Nothing here is required to use—read however you prefer.
